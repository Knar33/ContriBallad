ContriBallad generates a song based on Github contribution history.
The key/scale is determined by the first letter of the username (a = c minor, b = c major, c = c# minor etc.)
The color of each square determines the note/percussion played.
The day of the week determines the instrument:
  • Sunday: Piano, upper half of scale
  • Monday: Piano, lower half of scale
  • Tuesday: Percussion, Cymbals
  • Wednesday: Percussion, Snare drum
  • Thursday: Bass, 4 notes throughout the scale
  • Friday: Guitar, lower half of scale
  • Saturday: Guitar, upper half of scale
